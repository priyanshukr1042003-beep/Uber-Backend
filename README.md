🚖 Ride Booking Backend (Uber Clone)

A scalable backend system for a ride-booking platform inspired by Uber. Built with Node.js, this project demonstrates real-world backend concepts like asynchronous job processing, caching, and modular architecture.

✨ Features
RESTful APIs for rider and ride management
Background job processing using BullMQ
Redis-based caching and queue system
MongoDB integration with Mongoose
Schema validation using Zod
Centralized error handling
Request logging with Morgan
Modular structure (controllers, services, queues, workers)
🛠️ Tech Stack
Backend: Node.js, Express.js
Database: MongoDB (Mongoose)
Cache & Queue: Redis, BullMQ
Validation: Zod
Logging: Morgan
Env Management: dotenv
